# Harness Correctness Audit — Decision-Ready Report

## 1. Headline verdict: TRUSTWORTHY

**No past keep/kill verdict is invalidated by a harness bug.** The audit surfaced three CONFIRMED code-level defects and adversarially refuted two others, but every confirmed defect is either (a) on a code path the live gate no longer uses, (b) latent and never fired on real data, or (c) corrupts *attribution* rather than the accept/reject decision. ## 3. Rejected / not-a-bug (flagged, then refuted — audit was adversarial)

- **Intraday macro entry gates use same-day daily close (look-ahead)** — `intraday_agent_simulator.py:273`. Pattern is genuinely in the source, but **dormant on every verdict-producing path**: gates #1/#2/R5-C are off-by-default (flags=0/False); the one flag nightly retrain enables (`use_opportunity_score`) is inert because its `_spy_daily_closes` is only loaded when `pm_abstention_spy_ma_days>0`, which retrain_cron never forwards. Would only fire if a human explicitly set the flags. **REJECTED** — fix still worth landing as PIT hygiene.
- **Swing dispersion-gate p10 floor uses whole-fold (future) quantile** — `app/ml/dispersion_gate.py:77`. Genuine code smell, but reachable only via `--rebalance-dispersion-gate` (default False); zero uses in `logs/`, no verdict in DECISIONS/BACKLOG, and the default-True intraday dispersion gate is a *separate* inline implementation that never imports this module. **REJECTED** — never enabled in a WF/CPCV run.

## 4. MINOR cleanups (correctness nits, do not change a verdict)

- **cpcv DSR `unique_obs` divisor** (`cpcv.py:255`) — divides by multiplicity assuming SUM of fold obs, but path_n_obs is the MEAN → T understated ~44%. Only makes the *legacy* DSR gate stricter (can't manufacture KEEP); live ruler_v2 unaffected. Fix: set `_dsr_n_obs()` to `len(oos_returns_dated)`.
- **cpcv trained-expanding overlap-guard collapse** (`cpcv.py:1183`) — duplicate single-fold Sharpes deflate std / inflate path t-stat. Bypassed for rules-based sleeves (`is_trained=False`); live ruler_v2 ignores path t-stat. Fix: dedupe path stats by unique fold identity.
- **gates Calmar no-DD vol-floor** (`gates.py:135`) — uses `ann_vol/12` as "monthly vol" instead of `ann_vol/√12` → floor DD ~3.5× too small, inflates Calmar on no-DD folds (bias toward KEEP). Report-only under ruler_v2. Fix: `/math.sqrt(12)`.
- **intraday full-day SPY bars unsliced** (`intraday_agent_simulator.py:392`) — SPY overlay features read end-of-day; neutralized by cross-sectional demean except when <2 candidate symbols. Latent edge case. Fix: slice `spy_day.iloc[:entry_offset]`.
- **tail one_bet fails OPEN on undeterminable stress corr** (`tail_diagnostics.py:218`) — `np.nanmax` of two NaNs → reports "diversified". Needs <50 conditioning days; real book has >2500. Fix: surface UNDETERMINED instead of one_bet=False.
- **liquid_universe survivorship** (`futures_data.py:104`) — current-liquidity membership applied to all folds → forward-looking selection. **Already disclosed** in code comment + DECISIONS 2026-06-18; edge separately confirmed in modern regime. Keep as-is with the as-of rerun already partially done.
- **intraday WF embargo computed but never applied** (`walkforward_tier3.py:1316`) — inert today (frozen single model, no per-fold retrain; test windows disjoint). Fix or delete `_embargo_intra` before per-fold retrain is ever wired in.
